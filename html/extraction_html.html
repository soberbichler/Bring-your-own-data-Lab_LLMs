NuExtract Universal Document Extractor

Extracts structured metadata from any document type using a local LLM. No data leaves your computer.

Setup: In LM Studio, load NuExtract-v1.5 (or any instruct model). Start the server in Developer tab. Set temperature to 0.

Define what you want to extract. Use "" for single values, [] for lists. The model fills in the values from your document.

These teach the model by example. Edit to match your documents.

Longer texts will be truncated. NuExtract: 3000. Larger models: up to 8000.
