▸ Workshop tips
  • Use {observation} in the user prompt to insert the selected text.
  • Try the preset prompts — compare Descriptive vs Codebook to see how structure changes outputs.
  • A precise system prompt defines the model's role; the user prompt gives the task.
  • Enable Compare A/B to run two different prompts side-by-side on the same observation.
  • Ask: does the model "over-code" vague passages? Does adding definitions help?
  • Look for hallucinations: details the model adds that aren't in the source text.
